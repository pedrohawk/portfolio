FX Project: Factory management software

This project includes a MAUI application used as a backoffice, allowing managers to control and oversee all factory operations.

On the factory floor, a separate application is available for operators to log events related to each component they work on.

Additionally, an API handles the communication between the applications and the database, ensuring data consistency and integration across the system.

Stack: .NET 8, MAUI, Entity Framework Core, SQL Server
